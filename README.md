# Мафия — Android-приложение (Kotlin + Jetpack Compose)

Полный перенос консольной игры "Мафия" на Python в нативное Android-приложение.
Material 3, тёмная тематическая палитра, передача устройства по кругу между игроками — как в оригинале.

## Куда положить logo.png

Положите файл строго с именем **`logo.png`** (можно квадратный, минимум 512×512px) в папку:

```
app/src/main/res/drawable/logo.png
```

Он автоматически заменит:
- логотип на главном экране (круглая иконка сверху),
- иконку приложения на рабочем столе телефона (adaptive icon).

Сейчас там лежит временный плейсхолдер — просто перезапишите файл своим, имя и путь менять не нужно.

> Если хотите, чтобы иконка приложения смотрелась аккуратнее на разных лончерах,
> используйте PNG с прозрачным фоном и важным контентом по центру — края будут подрезаны адаптивной маской.

## Как открыть и собрать

1. Установите [Android Studio](https://developer.android.com/studio) (актуальная версия).
2. File → Open → выберите папку `mafia-app`.
3. Studio сама подтянет нужную версию Gradle (8.7) и зависимости — нужен интернет при первом открытии.
4. Подождите окончания индексации и Gradle Sync.
5. Запустите на эмуляторе или физическом устройстве кнопкой ▶ (Run 'app').

Минимальная версия Android — 7.0 (API 24).

## Структура проекта

```
app/src/main/java/com/dplusmop/mafia/
├── MainActivity.kt          — точка входа, навигация по экранам
├── model/
│   ├── Roles.kt             — 14 ролей: иконки, цвета, описания, раздача ролей
│   ├── Player.kt            — модели игрока лобби/партии
│   ├── GameEngine.kt        — вся игровая логика (ночь, голосование, победа)
│   └── GameViewModel.kt     — состояние приложения и переходы между экранами
└── ui/
    ├── theme/               — тёмная Material 3 палитра и типографика
    ├── components/          — переиспользуемые UI-блоки (кнопки, карточки)
    └── screens/             — экраны: меню, лобби, роли, ночь, голосование, финал
```

## Перенесённая логика игры

Всё взято 1:1 из оригинального Python-скрипта на `rich`/`colorama`:

- 14 ролей: мафия, дон, адвокат, доктор, комиссар, красотка, ангел, мэр, шут,
  ведьма, наркоман, маньяк, снайпер, мирный житель.
- Автоматическая раздача ролей по числу игроков (4–13 — готовые наборы, 14+ — процедурная генерация).
- Ночная фаза по очереди: наркоман → красотка → мафия (+адвокат) → доктор → комиссар → ангел → ведьма → маньяк.
- Блокировки от наркомана/красотки, скрытие Дона от комиссара, лечение себя доктором.
- Воскрешение ангелом (с 2-й ночи), одноразовый обмен ролями ведьмой, одноразовый выстрел снайпера днём.
- Дневное голосование с весом голоса мэра ×2 и пропуском голоса у "жертвы" красотки.
- Победа Шута при изгнании голосованием, победа мирных/мафии/маньяка по стандартным условиям.
- Финальный экран со списком всех ролей и статусом "жив/убит".
